# Obsidian-Mitre-Backbone
### An Obsidian notes offensive security note-taking template

<img width="871" height="792" alt="image" src="https://github.com/user-attachments/assets/3f679d40-9193-41d6-a870-9c33e16e2e26" />

Obsidian Mitre backbone POC is a flexible offensive security obsidian notes template used to map ideas to MITRE ATT&CK. In its current state, it is primarily intended for those doing CTFs and studying for certifications.

There are multiple options to generate the backbone. 
#### usage options:
- use python generators included
- use prompts included
- build your own prompts based off of mine
